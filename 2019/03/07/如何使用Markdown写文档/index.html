发表于2019-03-07|更新于2021-06-18|开发文档

现如今使用Markdown写文档越来越普遍，出现在各种开发文档，博客，开源项目中。为何这种语法会这么受欢迎呢？就是现在，请系好安全带，跟随新手司机一起来揭开它神秘的面纱！

初识markdown

Markdown是一种纯文本格式的标记语言。通过简单的标记语法，它可以使普通文本内容具有一定的格式。

特点

  • 轻量，纯文本语言。
  • 简单，相比传统编辑器操作简单。
  • 通用，只要支持Markdown都能获得一样的编辑效果，可以让作者摆脱排版的困扰，专心写作。
